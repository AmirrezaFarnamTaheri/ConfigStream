We believe that everyone has the right to access information without restriction. Our mission is to empower users worldwide with the tools they need to stay connected and informed, bypassing digital barriers and promoting a more open internet.

Our system is fully automated using GitHub Actions. We perform hourly retests of existing proxies to verify connectivity and demote or remove those that have gone offline. Every 3 hours, we fetch fresh proxies from public sources, deduplicate them, test each one for security and performance, add geolocation data, and generate updated configuration files—ensuring you always get working, tested configurations.

We take security seriously. Every proxy from our 240+ curated sources is subjected to comprehensive automated tests before being listed:

  • Content Injection Detection: Filters out proxies that modify page content or inject malicious code.
  • SSL/TLS Validation: Ensures secure HTTPS connections and validates certificate chains.
  • Header Preservation: Verifies proxies don't strip important security headers.
  • Port Scanning Prevention: Removes proxies with suspicious open ports.
  • Performance Measurement: Records latency and response times with intelligent caching (2-hour TTL) to help you choose fast, reliable proxies.
  • Batch Processing: Optimized pipeline with 750-proxy batches and concurrent testing for maximum efficiency.

This service is designed for bypassing restrictions, not for guaranteed privacy. Please be aware of the following:

  • Limited Privacy: Your traffic passes through third-party servers whose operators may log activity.
  • Variable Speed: Performance varies by location, server load, and network conditions.
  • No Guarantees: As a free, community-driven service, uptime and support are best-effort only.

This project thrives on community contributions with over 240 carefully curated proxy sources from trusted repositories. The sources are publicly available on GitHub, and we welcome contributions from developers to help us improve our testing methods, add new sources, and expand our reach. Our automated pipeline runs every 6 hours, continuously updating configurations with the latest proxies.
